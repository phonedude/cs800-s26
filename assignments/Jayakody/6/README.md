## Dissertation presented

**Title:** Large Language Models (LLMS) for Clinical Note Generation: International Classification of Disease (ICD) Code, Knowledge Graph (KG) and Prompt Evaluation  

**Author:** Ivan P. Makohon  
**Institution:** Old Dominion University  
**Department / degree:** Computer Science; Doctor of Philosophy (PhD)  
**Document type:** Dissertation  
**Date of award:** Fall 2025  

**Committee:** Yaohang Li (director); Vikas G. Ashok; Michele C. Weigle; Jian Wu; David S. Courson  

**Abstract (brief):** The work studies **chain-of-thought (CoT) prompt engineering** for **clinical note generation** with LLMs, using **ICD codes**, **patient context**, and **similar clinical cases** in prompts. It also discusses **knowledge graph** ideas in that setting. **Evaluation** uses **six cases** from the **CodiEsp** test set across multiple LLMs; CoT strategies are reported to improve over a **standard one-shot** prompt.

**DOI:** [10.25777/7vkp-y347](https://doi.org/10.25777/7vkp-y347)
