## Summary of the relationship of the two papers
The second paper has improvement from the first paper via a different method. The papers are talking about protein structure prediction. The first paper is using semi-flexible refinement on complex which allow high accuracy and fast processing. The second paper is using flexible refiement on single chain proteins which allow improved accuracy since loop and elbow issue resolved.
