# Dissertation Presentation – Election Security

##  Assignment Description
This project is part of a Research Methods course where students are required to:
- Select a dissertation in consultation with their advisor
- Analyze and understand the research work
- Prepare and deliver a **20-minute presentation**

The presentation focuses on:
- Research questions
- Technical highlights
- Results and evaluation
- Key contributions

This project is based on the dissertation:

**"A Framework for Securing Electoral Systems Through Addressing Cybersecurity Vulnerabilities in Modern Voting Infrastructure"**  
by *Jason M. Green*

---

##  Objective
The goal of this presentation is to:
- Understand cybersecurity risks in modern voting systems
- Analyze vulnerabilities in election infrastructure
- Evaluate proposed solutions such as honeynets and SIEM systems
- Present a concise and critical summary of the research

---

##  Key Topics Covered
- Election system architecture
- Cybersecurity vulnerabilities in voting systems
- Attack vectors (USB attacks, DDoS, remote access)
- SIEM (Security Information and Event Management) evaluation
- Honeynet-based defense mechanisms
- Nearest Neighbor Attack (NNA)
