## Description

This assignment requires writing a literature review of at least five research papers related to my research area. The goal is to demonstrate an understanding of the current state of research, summarize the contribution of selected papers, and synthesize how these works relate to each other and the broader research landscape.

The literature review includes the following sections:

- Introduction – Overview of the research area and motivation for selecting the papers.
- Summaries – Individual summaries describing the main contributions of each selected paper.
- Synthesis – Discussion of how the papers relate to each other and to the overall research direction.
- References – A list of references generated using BibTeX.

List of papers that are reviewed:

- [Paper 1](./papers/paper1.pdf): scBERT as a large-scale pretrained deep language model for cell type annotation of single-cell RNA-seq data
- [Paper 2](./papers/paper2.pdf): Transfer learning enables predictions in network biology
- [Paper 3](./papers/paper3.pdf): scGPT: toward building a foundation model for single-cell multi-omics using generative AI
- [Paper 4](./papers/paper4.pdf): Deciphering driver regulators of cell fate decisions from single-cell transcriptomics data with CEFCON
- [paper 5](./papers/paper5.pdf): scMamba: A Pre-Trained Model for Single-Nucleus RNA Sequencing Analysis in Neurodegenerative Disorders
